In this project , a simple chat application is done with the help of php, ajax, jquery and mysql.

Login and logout system using sessions has done for recognise different users using php.

Ajax is used to refresh data every moment behind the scene.

Jquery is used for display the message properly.

Mysql is used for store user information and messages.
